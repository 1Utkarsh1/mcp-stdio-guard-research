# Environment

The evaluation scripts are intentionally lightweight.

## Required

- Node.js 20 or newer
- npm
- Network access for `npx` when evaluating an npm-published
  `mcp-stdio-guard` version

## Recommended Metadata to Record

Each run manifest records:

- evaluated `mcp-stdio-guard` package spec
- Node.js version
- platform and architecture
- fixture command
- guard exit code and observed issue codes

Additional machine details can be recorded in external lab notes when needed.
